# Git commit count

1. command line

```
git rev-list --all --count
git rev-list HEAD --count
git rev-list --branches  --count
git shortlog -s
```

```
$ git rev-list HEAD  --count
1775
$ git rev-list --all --count
1816
```

2. git2rs

```
git2::Revwalk.count()
```

# commit which has not been pushed

```
git log --branches --not --remotes
```

# git log rev range branch

```
git log --oneline start..branch
```
